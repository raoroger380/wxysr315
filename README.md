# 🎂 伍希彦生日贺卡

一个精美的生日贺卡网站，为伍希彦定制。展现粒子文字变形、互动蛋糕场景、祝福卡片和焰火特效等创意功能。

## ✨ 功能特性

### 首页 (index.html)
- **粒子文字系统** - 动态点阵在不同文字形状间变形
- **互动蛋糕场景** - 点击或吹麦克风吹灭生日蜡烛
- **麦克风检测** - 基于音量阈值识别吹气动作，自动触发场景转换
- **祝福显示** - 蜡烛吹灭后显示温暖的生日祝福
- **惊喜链接** - 链接到隐藏的惊喜页面

### 隐藏页面 (jingxi.html)
- **祝福卡片** - 心形弹出式贺卡在屏幕上随机出现
- **参数化心形曲线** - 卡片沿优美的心形轨迹动画移动
- **焰火特效** - 烟火/烟花动画两种模式：
  - 普通圆形爆炸
  - 心形爆炸效果
- **物理引擎** - 火箭轨迹、重力衰减、粒子拖尾效果

## 🛠️ 技术栈

- **100% HTML/CSS/JavaScript** - 无依赖、无构建工具
- **Canvas API** - 粒子系统、焰火特效渲染
- **Web Audio API** - 麦克风吹气检测
- **SVG** - 文字形状光栅化
- **Google Fonts** - 标题字体（可选）

## 🎮 交互说明

### 首页交互
1. **等待动画加载** - 欣赏粒子文字变形效果
2. **吹灭蜡烛** - 两种方式：
   - 点击蛋糕上的蜡烛
   - 对着麦克风吹气（需允许麦克风权限）
3. **查看祝福** - 蜡烛熄灭后显示生日祝福
4. **进入隐藏页面** - 点击链接查看惊喜

### 隐藏页面交互
1. **祝福卡片** - 自动出现并消失
2. **焰火秀** - 所有卡片消失后自动触发精彩烟火秀
